# 🖼️ OCR to Speech Converter (Python)

This is a simple Python project that:
1. Extracts text from images using **OCR (Optical Character Recognition)**.
2. Converts the extracted text to speech using **Text-to-Speech (TTS)**.

Perfect for quickly reading printed text aloud from screenshots, scanned documents, or photos!

---

## 📋 Features

- 🔍 **OCR**: Detects and extracts text from images using `easyocr`.
- 🔊 **TTS**: Reads the extracted text aloud using `pyttsx3`.
- ✅ Works **offline** (no internet needed).
- 💻 Cross-platform: Works on **Windows, macOS, and Linux**.
